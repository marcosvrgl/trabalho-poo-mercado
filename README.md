## Objetivo do Projeto

O objetivo deste projeto é desenvolver um sistema simples de gerenciamento de mercado, aplicando conceitos de Programação Orientada a Objetos em Python.

O sistema permite o cadastro de clientes e funcionários, o gerenciamento de produtos em estoque e a realização de compras por clientes.

## Funcionalidades

O sistema possui as seguintes funcionalidades:

- Acesso de funcionário por senha;
- Cadastro de funcionários;
- Cadastro de clientes;
- Listagem de estoque;
- Adição de produtos ao estoque;
- Reposição de produtos existentes;
- Remoção de produtos;
- Compra de produtos por clientes;
- Atualização automática do estoque após a compra;
- Finalização da compra com escolha da forma de pagamento;
- Tratamento de entradas inválidas;
- Leitura e escrita de dados em arquivos `.txt`.
