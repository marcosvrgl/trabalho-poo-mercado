# Sistema de Mercado em Python

Projeto acadêmico desenvolvido para a disciplina de Programação Orientada a Objetos.

## Objetivo

O sistema tem como objetivo simular o funcionamento básico de um mercado, permitindo o cadastro de clientes e funcionários, o gerenciamento de estoque e a realização de compras por clientes.

## Funcionalidades

- Acesso de funcionário por senha
- Cadastro de funcionários
- Cadastro de clientes
- Listagem de estoque
- Adição de produtos ao estoque
- Reposição de produtos
- Remoção de produtos
- Compra de produtos por clientes
- Atualização automática do estoque após a compra
- Finalização da compra com escolha da forma de pagamento
- Tratamento de erros para entradas inválidas
